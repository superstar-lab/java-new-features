# Java 9 New Features

A project to explore more about the new features of Java...

* Modules
  * Creating modules
  * Exports and requires
  * Using a non-exported class through its exported interface
  * Running a module
* JLink


The shell script uses env var JAVA9_HOME to jdk-9 dir.
